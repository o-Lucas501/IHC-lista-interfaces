# 📘 Interação Humano-Computador (IHC) – Evolução das Interfaces

## 1. Contextualização

A Interação Humano-Computador (IHC) não surgiu com dispositivos modernos como smartphones. Ela é resultado de décadas de evolução tecnológica e conceitual, passando de interfaces baseadas em comandos textuais até experiências naturais e intuitivas.

Compreender essa evolução é essencial para evitar erros de design e criar sistemas mais eficientes, acessíveis e centrados no usuário.

---

## 2. Tema Escolhido

👉 Interface de Linha de Comando (CLI)  
*(Você pode trocar por GUI ou NUI, se necessário)*

---

## 3. Roteiro da Pesquisa

### A. Evolução Histórica

#### 👨‍🔬 Pioneiros e limitações
- As primeiras interfaces CLI surgiram nas décadas de 1960 e 1970.
- Usuários interagiam com o computador por meio de comandos de texto.
- Limitações:
  - Ausência de elementos visuais
  - Necessidade de memorização de comandos
  - Alta curva de aprendizado

#### 🎯 Problema de usabilidade resolvido
- Permitiu interação direta com o sistema
- Maior controle e precisão nas operações
- Automatização de tarefas via scripts

---

### B. Aplicação de Princípios de IHC

#### 💻 Sistema moderno analisado:
Terminal do sistema operacional (Linux/Windows PowerShell)

#### 🔍 Avaliação com base nos princípios de Nielsen:

**1. Visibilidade do status do sistema**
- O terminal mostra respostas imediatas após cada comando
- Feedback claro (sucesso, erro, execução)

**2. Mapeamento com o mundo real**
- Baixo para iniciantes (comandos técnicos)
- Alto para usuários experientes

**3. Prevenção de erros**
- Fraca: exige precisão na digitação
- Erros podem gerar falhas críticas
- Algumas melhorias modernas incluem autocomplete

---

### C. Estudo de Caso Comparativo

#### ❌ Design Ruim (CLI)
- Necessidade de memorizar comandos complexos
- Erros pequenos causam falhas grandes
- Pouco feedback visual

#### ✅ Design Bom (CLI moderno)
- Autocompletar comandos
- Histórico de comandos
- Mensagens de erro mais claras
- Sugestões automáticas

## ✅ Conclusão

A evolução das interfaces demonstra uma busca constante por maior usabilidade e acessibilidade. Cada modelo (CLI, GUI, NUI) resolve problemas específicos e ainda possui aplicações relevantes atualmente.

O entendimento desses paradigmas é fundamental para projetar sistemas mais eficientes e centrados no usuário.
